# Hello Oxygen.jl

1. Install [Oxygen.jl](https://github.com/OxygenFramework/Oxygen.jl).
```julia
import Pkg; Pkg.add("Oxygen")
```

2. Clone this repogitory.
```
git clone https://github.com/ohno/hello-oxygen.git
cd hello-oxygen
```

3. Start the server.
```sh
julia app.jl 8090
```

4. Visit http://localhost:8090.
